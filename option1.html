Setup Commands - Setup Commands Onturn Commands Offturn Commands
Command Syntax Description
Enter Battle battle Enter the current battle channel
Current battle status current Recieve report on any battles currently in progress
Player status points Recieve player identification, personal points, and current team affiliation and team points
Select Ship ship (slot#) (class) (id#) (shipname) Specify ship to use during a battle setup. Acceptable class identifiers:
Galaxy - Defiant - Steamrunner
Kvort - Vorcha
Warbird
Jemhadar - Jemcruiser - Galor
Peregrine - Miradorn - Daryyli - Husnock
Planet Status planet (planet#) Recieve status report over specified planet
Build Planetary Structure build (planet#) (#units) (structure-type) Adds structures to planet's build queue. Accepted structure types:
SLW -Surface Light Warehouse
SHR -Surface Hydrogen Reactor
SFP -Surface Fission Plant
SMS -Surface MineShaft
SVS -Surface VaporShaft
SAL -Surface Assembly Line
SRZ -Surface Residental Zone
SEC -Surface Education Center
SRL -Surface Research Lab
OPM -Orbital Payload Module
OSA -Orbital Solar Array
OVS -Orbital VaporShaft
OCM -Orbital Crew Module
Buy Resources buy (planet#) (#units) (unit-type) Buy resource units for a planet off the market. Unit types:
ORE
GAS
PARTS
Sell Resources sell (planet#) (#units) (unit-type) Sell resource units from a planet's stock to the market pool. Unit types:
ORE
GAS
PARTS
Labor Production labor (planet#) (percent#) Change the current labor setting for a planet's production percentage, a number from 0-100 (don't put in the % sign)
Market Status market Recieve market product price report
