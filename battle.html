Match Battle Simulation Rules
Each team starts with 160 points and 6 ship slots to fill.
Each team's CO/chosen match representative will roll a d30 within the battle channel:
-Lower Roll is awarded First Slot, and the odd numbered ship slots from 1-11
-Higher Roll takes even numbered ship slots from 2-12
160 + Team Points = Total points for use in match

Team Point Costs:
Ship point value- Purchase ship for combat
2 points- Add 1 extra Repair Team to a ship
2 points- Add 1 extra Security Team to a ship
3 points- Upgrade ship core shielding 1 level
1/10 ship point value- Upgrade ship evasive package 1 level (once per ship)
4 points- Upgrade one Photon/Plasma Torpedo launcher to Quantum Torpedo (must do all on a ship)
40 points- Research and design a new ship class (procedure to come soon)

Team Point Awards
Official Match:
Winning Match: +2 points
Ships killed/set adrift: 1/10 of base cost
Ships captured: 1/5 of base cost
Personal Experience Awards
Official Practice:
Member of Winning Team: +1 point
Member of Losing Team: No value
Each light class ship killed/set adrift: +1 points
Each medium class ship killed/set adrift: +2 points
Each heavy class ship killed/set adrift: +3 points
Each ship captured: +3 points
Warping away: -1 point
Getting Killed/set adrift: -1 points
Self-Destructing: -2 points

Mission:
Accomplishing Mission Parameters: +3 points
Opposing Team Fails to Accomplish Mission Parameters: +2 points
Failing to Accomplish Mission Parameters: -1 point
Opposing Team Accomplishing Mission Parameters: -1 point

Scheduled Battle:
Member of Winning Team: +4 points
Member of Losing Team: +2 points
Each light class ship killed/set adrift: +2 points
Each medium class ship killed/set adrift: +3 points
Each heavy class ship killed/set adrift: +6 points
Each ship captured: +5 points
Warping away: -2 points
Getting Killed/set adrift: -1 point
Self-Destructing: -4 points
